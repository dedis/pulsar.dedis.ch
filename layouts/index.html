Pulsar

Decentralized Verifiable Randomness

To retrieve and verify the latest random value using the cothority configuration, install Golang (v1.8+) and then execute:


    export X=github.com/dedis/cothority
    go get -u $X
    cd $GOPATH/src/$X
    git checkout randhound_659
    go get -u $X/randhound/pulsar
    wget https://pulsar.dedis.ch/public.toml
    pulsar random public.toml

Disclaimer: This service is still experimental. Do not use it in security-critical production systems. For more details on the underlying constructions, please refer to the accompanying research paper published at IEEE Security and Privacy 2017.
